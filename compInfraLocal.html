Local network infrastructure

Switches and Hubs

Ethernet networks use a design called star bus topology,which means that nodes are connected to or more centralized devices.A centralized device can be a switch or a hub.Each of these devices handles networks differently

Hubs

1. An ethernet hub transmits the message to every device except the device that sent the message.
2. A hub is just a pass-through and distribution point for every device connected to it,without regard for what kind of data is passing through and where the data might be going.
3. Hubs are outdated technology,it is replaced by switches.

Switches

1. A switch is a smarter and more efficient than a hub because it keeps a table of all MAC addresses for devices connected to it.
2. When a switch receives a message,it searches it's MAC address table for the destination MAC address of the message and sends the message only to the interface for the device using this MAC address.
3.Switch doesn't know all MAC addresses of devices connected to it,it keeps on learning about the MAC address as the message is a and records each source MAC address in it's MAC address table.
4. When it receives a message destined to a MAC address not in the table,the switch acts like a hub and broadcasts the message to all devices except the one that sent it

Two types of switches are managed and unmanaged switches.

  • Unmanaged switch
      Requires no setup or configuration other than connecting network cables to it's ports.It doesn't require an IP address and is appropriate for SOHO networks
  • Managed switch
      Has firmware that can be configured to monitor and manage network traffic.It's appropriate for large networks and can be used to manage QoS for prioritizing network traffic and to control speeds for specific ports can use a managed switch to subnet a large LAN into smaller subnets called virtual LANs( VLANs ) which can reduce network .Subnetting is done by assigning a group of ports on the switch to a different VLAN and directing the router as a DHCP server to assign a unique range of IP addresses to each vlan to create a subnet.Physical LAN is subnetted into two virtual LANs. With subnetting,broadcaast traffic is reduced because it is limited to each VLAN. The firmware on a managed switch is accessed through a browser using the switch's IP address,which is similar to how you access the firmware on a router.A switch requires an IP address only for the purpose of accessing it's firmware

    • Reasons you might add switch to your network:

      ∠ To add network connections A SOHO router has four to eight ports in a built in switch.When you need to add more connections,add switch in a location where you have multiple workstations or printers that need to be connected to the network.
      ∠ To regenerate the network signal An Ethernet cable should not exceed 100 meters( about 328 feet) in length.If you need to reach distances greater than that,you can add a switch in the line,which regenerates signal
      ∠ To manage network traffic Managed switches can be installed in strategic places on the network to subnet the network and manage network to improve performance

    Wireless access point and bridges
  • Routers can also be a wireless access point,wireless access point can be a dedicated device
  • A bridge is a device that stands between two segments of a network and manages network traffic between them
  • Bridge, access point helps to reduce the overall volume of the network traffic by not allowing messages across the bridge if it knows that the message addreeed to a destination on it's own segment.
  • Bridge doesn't know the MAC address just like switch,it learns the information by maintaining a table of MAC address from each message that arrives at the bridge
  • It learns which nodes are on which network segment and becames more efficient at preventing messages from getting on the wrong segment,which can bog down the network
    • Network Servers
  • Client computer contacts a server in order to request information or perform a task, such as when web browser connects with a web server and request a web page
  • Some servers are stand-alone devices but are often multiple network services are provided by a single server computer or servers might be embedded in other devices
  • Servers are sometimes embedded in router firmware such as SOHO router providing DHCP services or in an operating system such as web server capabilities embedded in windows server
  • Some network services are often provided by multiple servers,so that when one server goes down,the order server will take over and that is called redundancy
  • It will add reliability but it will also add complexity.Each time component is updated,any legacy technology on the network must be taken into consideration,which can result in a complex web of network server resources.

    • Client/server resources used on networks and the internet:

      ∠ Web server. A web server serves up web pages to client computer.Many corporates have their own web servers which are available privately on the corporate network.Other web servers are public,accessible from anywhere on the internet.Popular servers like Apache ( apache.org ) which runs on UNIX or Linux systems and Windows.the other server is Internet Information Services (ISS) which is embedded in windows server operating system
      ∠ Mail server Email is a client/server application which involves two mail servers SMTP is used to send email messages and either POP3 or IMAP4 is used to deliver an email message to a client.A
      ∠ File server. Stores files and makes them available to other computers.A network adnimistrator can make sure this data is backed up regularly and kept secure
      ∠ Print server. Manages network printers and makes them available to computers through out the network.Expensive network printers can handle high-capacity print jobs for many sources,eliminating the need for desktop printers at each workstation.Windows business and professional versions include the Print managent console which enables technicians to diagnose network printers from any computer within the network
      ∠ DHCP server. Leases IP address to a computer when it first attempts to initiate a connection to the network and requests an IP address.DHCP server is configured to pull from a range of IP addresses,which is called DHCP scope
      ∠ DNS server. Store domain names and their associated IP addresses for computers on the internet or a large enterprise network.DNS servers are responsible for name resolution,which happens when a client computer sends an FQDN to a DNS server and requests the IP address associated with this character-based name
      ∠ Proxy server.is a computer thwt intercepts requests that a client such as a browser,makes of another server such as web server.The proxy server substitudes itt's own IP address for the request using NAT protocols.It might also store data that is used frequently by it's clients.Proxy seerver can provide the contemnt that it has cached,sometimes acts as a router to the internet, a firewall to protect the network,filter for emails and to restrict internet access by employees to prevent them from violating company policies
      ∠ Authentication server. Authenticate users or computers to the network so that they can access network resources.Active Directory which is a directory service included in windows server,is often used for this purpose on a windows domain.Authentication server stores user or device credentials such as username and password,validates an authentication request and determines the permissions assigned to each user,device or group
      ∠ Syslog server. Is a protocol that gathers event information about various network devices such as errors,failures,maintenance tasks and user loggin in or out.The messages about the events are sent to a central location called Syslog server,which collects the events into a database.Some syslog servers can generate alerts or notifications to inform network administrators of problems that might need attention

    Unified threat management appliance
  • Router stands between local network and the internet to route traffic between the two networks
  • It can also serve as a firewall to protect the network
  • A next-generation firewall (NGFW) Combines firewall functions with antivirus/anti-malware functions
  • NGFW components might be installed on a dedicated appliamce,router,servers or cloud
  • NGFW device can offer Unified Threat Management (UTM) services. A UTM appliace is also called security appliance,network appliance or internet appliance,stand between the internet and private network,as does a router and protects the network

    • A UTM applicaince might offer these types of protections and services
    ∠ Firewall.The firewall software filters incoming and outgoing network traffic according to IP addresses,ports,the type of messages the traffic contains and how the message was initiated
    ∠ Antivirus and anti-malware software More advanced than what might be installed on a server or worksattion
    ∠ Identity-based access control loists Control the access of the users or user group and log and reportactivities of these users to reveal misuse,data leaks or unauthorized access to resources.This feature can be used to satisfy legal auditing requirements for detecting and controlling data leaks
    ∠ Intrusion detection system Monitors all network traffic and creates alerts when suspicious activity happens.IDS software can run on a UTM appliance,router,server or workstation
    ∠ Intrusion preventition system Not only monotors and logs suspicious activity,it can prevent the threatening traffic from borrowing into the system
    ∠ End point management server Also called an endpoint security management system,provides monitoring of various endpoint devices on the network,The service can unsure that application updates,the system will restrict the device meets the security requirements,which gives an additional layer of protection to other network resources.
    ∠ VPN The appliance can provide VPN to remote users of the network to access network resources

    Ethernet standatrds and cables

    Ethernet can run run at four speeds,each version of Ethernet can use more than one cabling method

      ∠ 10-Mbps Ethernet It was invented by Xerox Corporation in 1970 and later became known as Ethernet
      ∠ 100 Mbps Ethernet or Fast Ethernet Improved version of Ethernet,operates at 100 Mbps and uses copper cabling rated CAT-5 or higher.Fast Ethernet networks can support 10 Mbps so devices that run at either 10 Mbps or 100 Mpbs can coexist in the same network
      ∠ 1000 Mbps Ethernet of Gigabit Ethernet Ethernet operates at at 1000 Mbps (1 Gbps) and uses twisted-pair cable and fiber-optic cable.Gigabit Ethernet is the most popular choice for LAN technology,because it can use the same cabling and connectors as Fast Ethernet,a company can upgrade from Fast Ethernet to Gigabit without rewiring the network
      ∠ 10-Gigabit Ethernet Operates at 10 billion bits per second (10 Gbps) and uses fiber-optic cable.It can be used on LAN,WAN and MAN and it is good choice for network backbone
    Twisted-pair cable

    Three main types of cabling used by Ethernet are: twisted-pair,coaxial and fiber-optic
    ∠ Twisted-pair cabling uses pairs of wires twisted together to reduce crossover,which is interference that degrades a signal on the wire.It the most popular cable used for LAN and uses RJ-45 connector
    ∠ The cable comes in two varieties: unshielded twisted-pair (UTP) and shielded twisted-pair (STP) cable
    ∠ UTP is not expensive as STP and is used for LAN networks
    ∠ STP cable uses the shield or covering around each pair of wires inside the cable that protects it from electromagnetic interference caused by electrical motors,transmitters or high-tension lines. ∠ Shielded cost more than unshielded and it is only used when demanded

    Fiber Optic

    Fiber-optic cable transmit signal as pulses of light over glass or plastic strands inside protective tubing.Fiber-optic cones in two types:Single mode ( thin,difficult to connect,expensive and best perfomance ) and Multimode ( Most popular). Single mode uses a single path for light to travel through and multimode cable uses multiple paths for light.

    Both single and multiple fiber-optic cables can be constructed as loose-tube cables for outdoor use or tight-buffered cables for indoor or outdoor use.Loose-tube is filled with gel to prevent water from soaking into the cable and tight-buffered cables are filled with with synthetic or glass yarn,called strength wires,to protect the fiber-optic strands

    Powerline networking or ethernet over power (EOP)
  • HomePlug standards was introduced in 2001 called powerline networking or Ethernet over Power (EoP), uses powerlines in a building to transmit data
  • Powerline is used in a bulting where cabling or Wi-Fi cannot reach,it is simple to set up,not expensive and can run Gigabit speeds
  • If a building or apartment is sharing a phase (electrical signal) with another building or apartment,the data might leak and be intercepted by neighbor.
  • Power adapter offers 128-bit AES encryption that is activated by pairing the adapters to each other,you can install manufacturers utility on your computer to create key network

    • Consider the following issues powerline networking presents

      ∠ Powerline must be plugged directly into a wall outlet.Plugging a powerline adapter into a power strip or sergee protector or sharing an outlet with energy hog like space heater hinders the function of the device
      ∠ Powerline adapters might be large and cover both outlets on a single wall plate,olny few adapters offer a pass-through outlet
      ∠ Sometimes people forget to use encryption options and end up with unsecured network
      ∠ Distance degrades quality.Try keeping the two adapters as close on the same circuit as possible.Jumping circuits decreases signal strength

    When shopping for powerline adapters,consider these things

      ∠ Make sure the adapter you are considering is Home Plug cerified.The HomePlug Alliace group ( homeplug.org )
      ∠ Make sure the adapter is rated for the latest HomePlug AV2 speed standard for Gigabit class data tranfers
      ∠ If you have limited wall outlets,youmight need a powerline adapter that offers a pass-through outlet
    Power over ethernet (POE)
  • Power over Ethernet is a feature that might be available on high-end wired network adapters to allow power to be transmitted over Ethernet cable
  • You can place a wuireless access point,IP phones,webcam,or other devices that needs power in a position in a building where you don't have an electrical outlet.
  • The Ethernet cable to the device provides both power and data transmissions
  • PoE can provide up to 25.5 W from a single Ethernet port,even though the power keeps degrading with the length of the cable
  • Most high-quality switches provide PoE,if it doesn't, you can attach a PoE injector which adds power to an Ethernet cable
  • Some devices are designed to receive both power and data from Ethernet cable,such as webcam
  • Other devices,you must use a splitter that splits the data and power transmissions before connecting to the non-PoE device.
  • when setting up a device to receive power by PoE,make sure the device sending the power,the splitter and the device receiving the power are all compatible
  • Pay attention to the voltage and wattage requirements and the type of power connector of the recieving device
