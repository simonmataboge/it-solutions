Connecting a computer to a local network

Connected to an Ethanet wired local network

To connect a computer to a network suing an Ethernet wired or Wi-Fi wireless connectionn,do the following

  • 1. Network adapter and it's drivers should be installed before you can connect to any network.
  • 2. Do one of the following to connect to the network
    ∠ For wired network,Plug in the network cable to the Ethernet port,also called RJ-45 port.the port looks like large phone jack.An Rj-45 connector looks like Rj-11 connector but larger.Indicator light near the port should light up to indicate connectivity.For windows it should automatically configure the connection
    ∠For wireless, click Network icon in the taskbar and select wireless network.Click Connect.If the network is secured,you must enter the security key
  • 3. You will be asked if you want to make the PC discoverable if it's for the first time you connect.For private network,click Yes and for public click No
  • 4. Open your browser and make sure you can access the web.

For wireless connection,you can view the status of the connection,including seciruity key used to connect.Do the following

    ∠ Open Control Panel and open Network and sharing center OR right-click the Network icon in the desktop and click Open Network and Internet settings,click Change adapter settings or network and Internet, click Change adapter options

    ∠ In network connection right-click the Wi-Fi connection and click Status, In the wireless network properties box,select security tab.To view the security key,check Show charactors

If you have problem making a network connection,you can reset the connection.Open Network Connections and right-click the network.Select Disable .Right-click the connection again and select Enable. The connection is reset,you can try again to borwse the web.If you still can't accesss the web,it's time to dig deeper

Creating a VPN Connection

A virtual private network (VPN ) is often used by telecommuting employees to connect to the corporate netowrk by the way of the internet.VPN protects data by encrypting it from the time it leaves the remote computer until it reaches a server on the corporate network.the encryption technique is called a tunnel or tnneling .A VPN can be manged by operating systems,routers, or third party software such as OpenVPN ( openvp.net ).A VPN connection is a virtual connection which means you are setting up a tunnel over an existing connection to the internet.When creating a VPN connection on a personal computer,always follow instructions given by the network administrator who hosts the VPN.The company website might provide the VPN client software to download and install on your computer ,then you might be expected to double-click a configuration file to complete the VPN connection.Open VPN uses an .ovpn file for this purpose

    1. In the Network and Sharing center, Click Set up a new connection or network .Then select Connect to a workplace- Set up a dail-up or VPN connection to your workplace and click Next.

    2. In the Connect to a workplace dialog box,click Use my internet connection (VPN). In the next box,enter the IP address or domain name of the network.Your network administrator can provide this information.Name the VPN connection and click Create

To access VPN connection,click Network icon in the taskbar.click VPN connection and click Connect.Enter your user nake and password and click OKYour username and password are likely to be the same as work ID and passowrd to your account on the windows domain on the corporate.

Problems connecting to a VPN can be caused by the wrong authentication protocols used when passing the user name and password to the VPN.To configure these settings, go to the Network and Sharing Center and click Change adapter settings. In the Network Connections window,right-click VPN Connection and click properties In the properties box,select the Security tab.Select security settings for the type of VPN,encryption requirements and authentication protocols given to you by by the network administrator

Creating a Dial-Up Connection

Here are steps you need to set up and support a dial-up connection:

  • Install internal or external dial up modem in your computer.Make sure device manager recognises the card without errors
  • Plug the phone line into the modem prot ion your computer and into the wall jack.Phone lines use RJ-11 connector,which are the same connectors used for wired telephone
  • Open the Network and Sharing Center windows, click Set up new connection or network.In the box,select Connect to the Internet-set up a broadband or dial up connection to the internet and click next
  • Click dial up ,enter the phone number to your ISP you ISP username and password,and the name you decide to give the dial up connection

If dial-up connection won't work,here are are some things you can try

    ∠ Is the phone line working? Is the phone cord securely connected to the computer and the wall jack? .Plug in a regular phone and chevk for a dial tone
    ∠ Does the modem work?. Check device manager for reported errors about the modem.Does the modem work when making a call to another number (NOT your ISP)?
    ∠ Check the dial-up connection Properties box for errors.Click Change adapter settings in the Network and Sharing Center and then right-click the dial-up connection and select Properties from the shortcut menu.Is the phone number correct? Hasa 1 been added in front of the number by mistake? Does the number need to start with +27 to get an outside line?
    ∠ Try dialing the number manually from a phone.Do you hear beeps on thee other end? Try another phone number
    ∠ When you try to connect,do you hear the number being dialed?
    ∠ Try removing and reinstalling the dial-up connection

Dynamic and Static IP configuration

Computers use the IP address to find each other on the network.An IP address is assigned to a network connection when the connection is first made and can be:
∠ A 32-bit string,written as a four decimal numbers called Octets and separated by colons such as 192.168.8.102
A 128-bit string,written as eight hexadecimal numbers separated bt colons,such as 2001:0000:B80:0000:0000:D3:9C5A:CC
Msot networks use 32-bit IP addresses,which are defined by IPv4 (Internet Protocol version $). Some networks use both 32-bit addresses and 128-bit addresses which are defined by IPv6 (Internet Protocol version 6)

A Host is any device, such as desktop,laptop,printer on a network that requests or serves up data or services to other devices.To communicate on a network or the internet, a host needs this TCP/IP information:
∠ It's own IP address_192.168.8.102
∠ A Subnet mask,which is four decimal numbers separated by periods_255.255.255.0. When a computer sends message to a destination computer ,it uses it's subnet mask to decide whether the destination computer is on it's own network or another network
∠ The IP address of a default gateway.Computers can communicate directly with each other on the same network.When a computer sends a message outside the network,it will need to use default gateway to connect to the internet.For small businesses and home users,default gateway is a router
∠ The IP addresses of one or more DNS (Domain Name System or Domain Name Service) servers.Computers use IP addresses to communicate but people use computer names,such as www.mwserve.co.za to address a computer.When you enter www.mwserve.co.za in your bowser,your computer must find the IP address of the www.mwserve.co.za web server and does so by enquiring a DNS server.DNS servers can access databases spread all over the internet that maintain lists of computer names or domain names such as www.mwserve.co.za and it's IP address assignments.This group of databases is called the internet namespace and finding their IP address for a computer is called name resolution In www.mwserve.co.za,the name mwserve is the domain name

IP address,subnet mask,default gateway can be manually assigned to a computer's network connection.The computer's IP address is called static address.However,the process can be done automatically the first time a computer connects to the network,the IP address it receives is called a dynamic IP address, and the server that assign the address from a pool of addresses maintains is called DHCP (dynamic Host Configuration Protocol) server.A computer or other devices that connects to a network,request the IP address from DHCP server,it is called DHCP clientThe client is leasing an IP address.DHCP server that serves up IPv6 addresses is called a DHCPv6

Most networks use dynamic IP addressing by default.Windows request dynamic IP configuration from the DHCP server and there is nothing for you to configure,but a computer must have a Static IP address,so as an IT technician,you need to know how to configure static IP addressing.
Follow these steps to configure a static IP addressing:

  • Open the Network connections window, right-click the network connection and click Properties. In properties box on the networking tab,selct Internet Protocol Version 4 (TCP/IP) and click Properties.The Properties box appesrs
  • Dynamic IP addressing is used by default,Select Use the following IP address. then enter the IP address,submet mask and default gateway
  • If your network administrator has given you the IP addresses of DNS servers,Select Use the following DNS server addresses and enter up to two IP addresses.If you have additional DNS IP addresses,click Advanced and enter them on the DNS tab of the Advanced TCP/IP settings box.

Alternate IP address cofiguration

Traveling employee who uses laptop and his work network uses static IP addressing,even though most public networks use dynamic IP addressing.For travelling,configure the laptop to use dynamic IP addressing in order to connect to the public networks.However when a computer needs to connect to corporate network,it will need static IP address.the solution is to create an alternate configuration that the computer will use only if needed.

To create an alternative Cofiguration,use the general tab of the TCP/IPv4 Properties box to set the configuration for dynamic IP addressing,then click Alternative configuration tab,by default windows use sets an APIPA (Automatic private IP address) which begins with 169.254,when it cannot find DHCP server.Select User configured.then enter static IP address,subnet mask,default gateway and DNS server addresses for the alternate configuration to be used on the company network.Click OK and close all boxes.If the computer can't find DHCP server,it will use the new static IP settings you just entered

Managing network adapters

A computer makes a wired or wireless connection to a local network by the way of network adapter,which might be embedded on the motherboard or a network interface card (NIC) installed in an expansion slot on the motherboard.The adapter might be an external device plugged into USB port.It might also be called network adapter

Here are network adapter's features you need to be aware of:

  • The drivers a NIC uses NIC usually comes with drivers on CD or drivers can be downloaded from the web.After installing a NIC,you will install it's drivers.Problems with the network adapter can sometimes be solved by using device manager to update the drivers or uninstall the drivers and then install them
  • Ethernet speeds For wired networks,four speeds for ethernet are 10 Mbps,100 Mbps 1 Gbps and 10 gpbs.Most networks use Gigabit ethernet and support the two slower speeds.To see the speeds the NIC supports, open It's device manager .Select Advanced tab.In the list of properties,select the speed and Duplex. Full Duplex sends and receives transmissions at the same time.Half duplex works only in one direction at the time.Select Auto Negotiation for windows to use the best possible speed and duplex for a particular connection
  • MAC address Every NIC (wired or wireless) has a 48-bit (6-byte) identification number called MAC address or physical address,hard-coded on the card by it's manufacturer.The MAC address is unique for that adapter and is used to identify the adapter on the local network.Example of a MAC address is 00-0C-4E-AB-A5.
  • Status indicator lights. A wired network adapter might provide indicator indicator on the side of the RJ-45 prot that indicate connectivity and activity.Be sure to check the stsus indicator lights to verify that you have connectivity then the problem is related to hardware.Next,check the cable connections to make sure they are solid
  • Wake-on-LANA NIC might support wake-on-LAN which allows it to wake up the computer when it receives certain communication on the network.To use this feature,it must be enabled on the NIC.Open the NIOC's Properties box in device manager and click advancedtab.Make sure Wake on Magic Packet and Wake on Pattern match are both enabled.For on-board NIC,you must also enable Wake on LAN in BIOS/UEFI setup.Reboot the system,enter BIOS/UEFI setup and look for the optiion on a power management screen.It is not recommended that you do not enable Wake on LAN for wireless network adapter
  • Quality of Service (QoS) .NIC adapter feature,is the ability to control which applications' traffic have priority on the network.The feature must be enabled and configured on the router,enabled and configured on the router,enabled on the network adapters and configured in windows for every computer on the network that uses the high priority applications.To enableQOS on computers NIC,open device manager,on the Properties box in device manager,on the Advanced tab,make sure Priority Enabled or Priority and VLAN Enabled is selected.If the option is not listed,then the NIC doesn't support it
