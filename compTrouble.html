    A good technician builds a strong network of resources that he or she can use when solving computer problem.Here are some resources to help you get started with your own list of reliable and tested sources of help:

  • The web Do web search on an error message,a short description of the problem or model and manufactuurer of the device might help.Check the website of the manufacturer or support forum of that device.It's likely that other technicians experienced the problem and posted it online,if you can't find the solution,post a new question on the web and other social platforms but be careful,not all technicians are correct or their advice is well intented
  • Chat,forum and email technical support Support from hardware and software support can be very helpful.Most techical support is available online via chat sessions during working hours.Support from manufacture is considered the highest authority for the correct fix to a problem
  • Manufacturers diagnostic software Many hardware manufacturers provide diagnostic software which is available to download from their websites .Toolkit is a good example that you can download in order to backup data,SeaTools is for windows or SeaTools for USb( to create bootable USB drive) and the use of the software to diagnose and fix problems with Seagate drives.Search for the software in manufacturers website and guidelined for using it
  • User Manual Often offers error messages, their meanings and how to fix the problem the,might also contain a list of diagnostic tools available
  • Technical associates in your organisation Don't be shy to ask for advise when you are stuck,There might be time where you have to turn the problem to a more experienced technician or someone who has access to more resources(This problem is called escalating the problem
    • Rule :1 Approach the problem systematically

    When trying to solve a problem,start at the beginning and walk through the situation,carefully.This rule is invaluable. Remember it and apply it all the time.if you can't find the explanation of the problem,go through the same process again also if you are faced with more than one problem in the same computer,fix one problem at the time,trying to fix many problems at the same time,it could be confusing

    Rule: 2 Establish your priorityies

    Decide what your first priority is , it might be to recover data lost or getting the computer running as soon as possible,Ask the user or customer to help you with making the decision, most users prefare data first unless they have a recent backup

    Rule :3 Beware of the user error

    Remember that many problem stem from the user error.Ask the user what the problem is and watch what the user is doing. Be careful though because some people don't like to be told they made a mistake(

    Rule :4 Keep cool and don't rush

    In some instances the customer or user might put you in corner and rush you to do the job,put you in a state of emergency,Never aloow yourself to rush.Look and think carefully before you leap.A wrong move can be costly,the best advise is not to hurry,plan your moves,do research and don't hesitate to ask for help

    Rule :5 Make no assumptions

    Documentation sometimes can be wrong,do your own investigation.if the user tells you the computer can't boot,boot the computer yourself to see if it really can't boot

    Rule 6: Try the simple things first

    Sometimes th esolution could be very simple and yet overlooked because we expect the problem to be difficult.Most problems are simple, check the siple things such as is the power cord connected or cable is not loose

    Rule 7: Became a researcher

    Take advantage of every resource available,documentation,online help,technical support and books.what you learn is what you take to the next problem,you are always learning something new everyday

    Rule 8:Divide and conquer

    Isolate the problem,if you try something and it is not working,reverse the situation of fixbefore trying soething else.as you divide bigger problems into smaller components,,you can analyse the componentseparetely

    Rule 9: Write things down

    Keep good notes as you are working,Draw diagrams,Clearly write dwon what you are learning,if you need to let go and get fresh air,it's easy to forget things when you come back

    Rule 10: Don't assume the worst

    never assume data is lost,there is always alot that can be don eot recover the data,if you think data on your hard drive is lost,do not write anything on that drive as you might write on top of that data,eliminating the chances of recoving data

    Rule 11: Reboot and start over

    Fresh starts are always good.Take a break,get away from the problem and start over

    Rule 12: Use the least invasive solution first

    Always remember you don't need to make things worse,so you should use the least invasive solution first.Don't format the drive until you have tried to fix the the problem without erasing everything on the drive Another example is,don't reinstall software until you tried to applying patches to the existing installation

    Know your starting point

    Find out what is working and what is not working before you take anything apart or try to fix anything.

    Troubleshooting the electrical system

    Electrical problems can occur before or after boot and can be consistent or intermittent.This is going to help you be awere of the symptoms of electrical problem so that you can be quick to diagnose the system without first suspecting Hard drive, OS, memory as the source of the problem

  • The computer appears "DEAD" no lights no spinning drive or fan
  • Computer sometimes locks up during boot,after several times it is successful
  • Error codes or beeps occur during booting but they come and go
  • You smell burnt parts or odour(Not a good sign)
  • Computer power down at unexpected times
  • Computer appears dead but you hear a whinning coming out of the power supply

    • The following is a list of things you can do to try and solve electrical system without opening a computer
    Try the simple things first

  • If you hear a burnt component,turn off the system and replace the burnt component
  • When you haer a whinning coming from the power supply,the power supply might be inadequate for the system or there might be a short.Don't press power buttoon to start up the system.Unplug the power cord so that the power supply won't be damaged,then open the case and find the short,if there is no short,replace the power supply
  • Is the power cord plugged in? Is the device turned on?
  • Are any cable connections loose?
  • Is the power cord controlled by a switch? turn it on
  • Is the circuit breaker blown? Is the house circuit overloaded?
  • Are all the switches on the system turned on? like computer, UPS, serge suppressor, monitor
  • Is there a possibility that the system has overheated
  • Check for source of EMI such as interference of fluorescent lightning or electrical fan of copier sitting near a computer

    • If the problem still not solved,it's time to open the case

  • Check all cables are connected from the power supply to the motherboard,check front panel is in place and all cards are seated securely
  • If you smell burned parts,search for a short and for the frayed and burnt wires.Disassemble the parts until you find the damaged parts
  • If you suspect the power supply,test it with a power supply tester

    • Troubleshooting motherboard, Processor, RAM

    If you suspect a bad component,substitude the bad one with a good one,be cautious though,if you suspect the power supply,check voltage coming out of the power supply before putting a new motherboard

    The following symptoms can indicate that motherboard,processor or memory module failing:
  • The systems begins to boot then powers down
  • No error message is displayed during the boot(Investigate the message)
  • The system reports less memory than you installed
  • The system becomes unstable,hangs or freezes at odd times. (This can have multiple causes,like power supply,RAM, hard drive,motherboard or processor,windows errors and overheating)
  • Intermittent windows or hadrive errors occur
  • Components on the motherboard or device connected to it don't work

    • The field replaceable unit(FRU) on a motherboard are the processor,the processor cooler assemly,RAM,and the CMOS battery,motherboard is also considered an FRU.When you troubleshoot the motherboard and discover that some components is not working,such as network port,you might disable that component in BIOS/UEFI settings and install a card to take it's place

    Troubleshooting principle Check the simple things first The motherboard and power supply are expensive and replacing them can be time consuming.Make sure you know the problem is then you can eliminate the components as the source of the problem.
    Follow these steps to eliminate windows,software,RAM,BIOS/UEFI settings and other software and hardware components as the source of the problem:

  • If an error message appears,Google the error message,Pay attention on the motherboard or processor manufacture or manufacture website
  • Problem might be virus.run a current version of antivirus software to check for any viruses
  • Memory module might be failing.Use memory diagnostics tools to test memory.
  • Problem caused by an application.Device manager is the best tool to use to check for potential hardware problem
  • Check event viewer logs for a record about the hardware or application problem
  • Download and install windows updates or patches,they might help you solve hardware or application problem
  • Ask yourself when did the problem start,was it immediately after installing a new device or or a new application,unistall it
  • A system does not have enough RAM can sometimes appear to be unstable.Consider upgrading RAM
  • The BIOS/UEFI might be corrupted or have wrong settings,check BIOS/UEFI setup,Is the CPU speed set incorrectly or is it overclocked? Reset BIOS/UEFI to default settings
  • Disable any quick booting features in BIOS/UEFI to get through to report of POST ,then look for errors reported on the screen during the boot
  • Flash BIOS/UEFI to update the firmware on the board
  • Check motherboard manufacturer's website for diagnostic software that might identify any problems with the motherboard
  • Update all drivers of motherboard components that are not working.if USB prots are not working,try to update USB drivers from the motherboard manufacture
  • If an onboard port or device isn't working but motherboard is stable,follow these steps
      1. Verify that the problem is not with the device using the port.Try moving the device to another system
      2. Go into BIOS/UEFI setup and verify that the port is enabled
      3. Check device manager and verify that windows recognise the device or port with no errors
      4. Try to update motherboard drivers for this device from the motherboard manufacturer's website
      5. If you have a loopback plug,use it to test the port
      6. If the problem still not solved,disable the port in BIOS/UEFI setup and install an expansion card to provide the same type of port or connector
  • Suspect the problem might be caused by a failing hard drive
  • Suspect the problem might be caused by overheating
  • Verify that the installed processor is supported by the motherboard.
