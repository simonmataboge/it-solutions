Strategies to troubleshooting any computer problem

Step 1: Interview the user and back up data

The first thing you do as technician is interview the user or customer if they are available.Ask questions to identify the problem,how to reproduce it and possible source of the problem,also ask if there is back up on of data that's in the hard drive.
Here are some questions that can help you as much as you can about the problem and it's root cause;

  • Please describe the problem,What error messages,unusual displays or failures did you see
      I see this blue screen with a message it makes no sense to me
  • When did the problem start? Does the computer have history of the same problem?
      When I first booted,after I loaded a beautiful picture on the website
  • What was the situation when the problem occured
      I was trying to start up my laptop or I was opning microsoft word or I was on the web search
  • What program or software were you using
      I was using internet explorer
  • What changes have resently been made to the system, like did you install a new hardware or software or update any thing
      Yes I moved the system from the launge to my room
  • Has there been a recent thunderstorm or electrical problem
      Yes,Last night, then I tried to to turn it on this morning
  • Have you made any changes on the system,hardware,software configurations recently
      No but I think my sister might have
  • Has someone else used the computer
      Sure,My son uses it all the time
  • Is there some valuable data on the system that is not backed up yet that I should know aboutb efore I start working
      Yes,my recent paper work, I did not save it on my USB drve
  • Can you show me how to produce the problem
      Yes,Let me show you

    • After you have spoken to the user,be sure to save any data that is not backed up before you start working on the computer Here are three options

    Here is a list of essential tools

      ∠ Use file explorer to copy data to another system If the computer works enough to boot to the windows desktop,you can use file explorer to save data on USB or external hard drive or to another computer
      ∠ Move the hard drve to another system If the system can't boot up,Don't do anything to jeopadise data, let it be the user's decision to do so,consider moving the hard drive to another computer
      ∠ Hire a professional file recovery service If your data is extremely valuable and other methods didn't work, you might consider hiring a professional recovery service,they are expensive but if your data is important,it will be worth it.To find the service you can search on the internet "data recovery",be sure you read reviews,understand the warranty and guarantees and perhhaps get recommendations from a satisfied customer
    Step 2: Examine the system and making your best guess

    Follow this process to make your best guess (best theory) and test it

  • 1. Reproduce the problem yourself and observe what the user has described If the user tells you the system is dead,plug it and test it,listen to fans, look lights and error messages.As you work on the system,refrain from making any changes until you come up with a theory for the source of the problem
  • 2. Decide if the problem is hardware or software related Make your best guess,if the system won't boot,it could be hardware,if the user tells you it hasn't worked since thunderstorm,it could be electrical.If explorer won't open even after the system load the windows,it could be software related.If word documents get corrupted,it means the user is not saving documents correctly or the application or OS might be corrupted,computer might have virus,hard drive might be failing.Investigate and decide if the problem could be hardware or software or the user
  • 3. Make your best guess and don't forget to question the obvious Here are some practical examples of question the obvious
      ∠ The video doesn't work.Your best guess could be a loose cable or monitor not turned on
      ∠ Excel worksheets getting corrupted.the user is not saving the workbook correctly
      ∠ The DVD is not reading the DVD.The DVD could have scratch
      ∠ The system doesn't boot and the message is hard drive not found.internal cables to the hard drive could be loose
      ∠ Application, Operating system,hardware support must be available online
      ∠ Verify the system,or any application changes by reffering to application events logs : These logs can be viewed using windows event viewer
      ∠ Find technical documentation available,if you can't find it,search on manufacturer website
      ∠ Use search engines to help you,include error message on the search,chances are someone might have posted the problem online
    • Step 3: Testing your theory

    Guidelines,simptoms and source of the problem

    Symptoms or Error messages What to do about the problem
    System shut down unexpectedly Try to find out what was happening at the time of shut down to pinpoint an application or device causing the problem.Possible sorces of the problem are overheating.faulty RAM,motherboard or the processor
    System shut down unexpectedly and starts back up Check the system overheating.Processor fan is working properly.Go to BIOS/UEFI to check temparature of the processor
    Blue screen with an error message (blue screen of death) The problem is caused by devices, drivers or corrupted windows installation.Begin by searching the error message on the web
    Black screen with an error message Caused by an error at POST,troubleshoot the device in the message
    System freezes without message Problem is likely caused by an application not responding.Open windows task manager and close the application
    POST code beeps One or no beep indicates that all is well after POST,but startup BIOS/UEFi communicates POST error.Search manufacture of the motherboard to interpret series of beep codes,Restart the system to count the beeps
    No power If you see no lights on the case,hear no fan spinning,make sure there is power in the system
    Blank screen when you first power up the system and no noise or lights Is power getting to the system?If YES make sure ppower supply is connected correctly and and connectors are securely connected
    Blank screen when you first power up the system and you hear noise and see lights Is the monitor on? Is data cable securely connected,is indicator light on the monitor on
    BIOS/UEFI looses it's time aand date settings "CMOS battery low" error message appears during the boot The battery is failing,replace the battery
    System reports less memory than you installed Memory module is not seated correctly or has failed.Troubleshoot memory
    System attempts to boot to the wrong boot device Go to BIOS/UEFI and change the boot device priority order
    Fan spin but no power to the device Check power supply,is power connector securely connected,Use power supply tester to check voltage outputs
    Smoke or burning smell Serious electrical issues,unplug the computer immediately
    Loud whinning noise The noise is bacause of power supply or failing hard drive
    Clicking noise Indicates the magnetic hard drive is failing.Replace the hard drive
    Intermittent device failure Could be caused by failing RAM or overheating,motherboard,processor or hard drive.Check processor temperature for overheating or check RAM for errors and run hard drive diagnostics
    Distended capacitors Failed capacitor on the motherboard or other circuit board are sometimes distented and discolored on the top of the capacitor.Replace motherboard
    Error appears during boot: Intruder detection system An intrusion system installed on the motherboard detected that the case was opned.Suspect a security breach
    Error appear during boot:Overclocking failed.Please enter setup to recofigure your system Overclocking should be disconnected.but it can also occur because of power supply failer
    Possible error messages:No boot device available,
    hard drive not found
    Fixed disk error
    Invalid boot disk
    Inaccessible boot device or drive
    Invalid drive specifications    		 Start up BIOS/UEFI did not fingd a device to use to load operating system.Make sure the boot device priority order is correct in BIOS/UEFI setup.Try booting from a bootable USB flash drive or DVD
    Possible error messages:
    Missing operating system
    Error loading operating system    		 Windows startup programs are missing or corrupted
    Continous reboot
    Beeps during POST Description
    Short beep or no beep The computer passed all POST tests
    1 Long beep and short beeps A video problem,no video card,bad video memory
    Continous beep A memory error or a loose card
    1 Long beep and 1 short beep Motherboard problem
    1 Long beep and 3 short beeps A video problem
    3 long beeps A keyboard controller problems
    Continous 2 short beeps and then pause Video card problem
    Continous 3 short beeps and then pause Memory error
    8 beeps followed by system shut down The system has overheated
    Continous high and low beeps CPU Problems
    Step 4: Planning your solution and then fixing the problem

    Replacing hard drive and motherboard can be expensive and time consuming,You need to know and plan what you will need to do in order to find the solution
    Do the following to plan your solution and fix the problem

  • Consider different solutions ansd select the least invasive one.Talk to the owner or user about the best solution
  • Do your best to determine what works and does't in the system before you apply your solution
  • Fix the problem.This might be as simple as plugging up a new monitor or might be as difficult as reinstalling windows and application software and restoring data from back ups
    • Step 5: Verifying the fix and taking preventive action

    After you have fied the problem, reboot the system and verify that all is in order.Have the user to verify that everything is working correctly,if you find another problem,return to step 2 in the troubleshooting process to examine the system and form a new theory as to the cause of the problem.After the user have verified that all is working,ask yourself "could this problem been prevented.If so, go extra mile to instruct the user how to prevent it,install updates and other measures to prevent the problem

    Step 6: Documenting what happened

    Good documentation helps you take what you have learnt into the next troubleshooting situation,train others,develop effective preventive maintenance and satisfy any audits or employer queries about your work.Be sure to include symptoms,the source of problem,your troubleshooting steps and what you did to fix it.Make the notes detailed enough so that you can use them later when solving the similar problems
    For on-site support,customer expects documentation about your services.Include in the documentation sufficient details broken down by cost of individual parts,hours worked and cost per hour.Give the documentation to the customer at the end of the service and keep copy for yourself.For phone support,documentation stay in-house
